Documentation

Bienvenue dans la documentation de Style.css

Installation

Pour utiliser Style.css, vous avez deux options :

Télécharger le fichier CSS

Vous pouvez télécharger le fichier CSS de Style.css et ses dépendances pour l'inclure dans votre projet.

<link rel="stylesheet" href="style.css">

Utiliser le CDN

Vous pouvez également utiliser le CDN de Style.css pour inclure le framework dans votre projet.

<link rel="stylesheet" href="https://untypequicode.github.io/style.css/css/style.css">

Les possibilités de Style.css

La gestion des sections

Accéder au style des sections

Télécharger le fichier CSS
<link rel="stylesheet" href="section.css">
Utiliser le CDN
<link rel="stylesheet" href="https://untypequicode.github.io/style.css/css/section.css">

Gestion des tailles

Vous pouvez gérer la taille des sections en utilisant les classes suivantes :

<section class="s-mh-25"></section>

La valeur de la taille est exprimée en pourcentage de la hauteur de la fenêtre.

Les tailles disponibles sont :

  • s-mh-25
  • s-mh-50
  • s-mh-75
  • s-mh-100

Gestion des placements du contenu

Vous pouvez gérer le placement du contenu des sections en utilisant les classes suivantes :

<div class="sv"></div>

Les placements disponibles sont :

  • sh
    : horizontal
  • sv
    : vertical

La gestion des animations

Accéder au style des animations

Télécharger le fichier CSS

Utiliser le CDN
<link rel="stylesheet" href="https://untypequicode.github.io/style.css/css/animation.css">

Gestion des animations

Vous pouvez gérer les animations des éléments en utilisant les classes suivantes :

<div class="ah-n"></div>

Le paramètre n permet de définir le type d'animation. La valeur de n est celle par défaut, il n'est pas nécessaire de l'ajouter.

Le paramètre h détermine que le composant sera caché au chargement de la page. Il faut ajouter la classe avec le s pour afficher le composant au chargement de la page.

Les animations disponibles sont :

  • ah ah-n
    : animation normale cachée
  • as as-n
    : animation normale affichée au démarrage à définir en plus de ah

Gestion des textes

La gestion des tailles de texte est automatique. Vous pouvez gérer la taille du texte le plus petit et le plus grand en utilisant les lignes de code css suivantes :

:root { --max-font-size: 15rem; --min-font-size: 2rem; }
